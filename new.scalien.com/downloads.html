The latest version of ScalienDB is 0.9.9.

Download the latest version from our github repository at http://github.com/scalien/scaliendb. 

  $ git pull git://github.com/scalien/scaliendb.git scaliendb
  $ cd scaliendb
  $ make
  $ make pythonlib # optional

The web management console is located in webadmin/index.html.

Don't forget the check out the documentation.
